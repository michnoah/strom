Lightweight, Cross-platform BitTorrent client designed for performance.

Features

  • BitTorrent extensions
    • DHT — Distributed Hash Table
    • PEX — Peer Exchange
    • Magnet Links
  • UPnP support
  • Hardware-assisted hashing
    Android: NEON SIMD instructions on ARMv7 and ARMv8 CPUs
    macOS: CommonCrypto
    Windows: BCRYPT
  • Memory mapped file I/O
  • Zero-copy memory architecture
    • Download: Network buffer → mmap disk write
    • Upload: mmap disk read → network buffer
  • Complete protocol rewrite in a memory-safe language
  • Optimized CPU and memory footprint for mobile devices
  • Modern minimal UI
